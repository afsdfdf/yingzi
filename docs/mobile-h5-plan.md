# 用户端 H5 优化计划

## 阶段目标

将当前移动端从“可演示”提升到“可上线的一期版本”，优先完善首页浏览、商品详情、购物车、订单和个人中心。

## 第一阶段：界面和信息架构

1. 统一顶部搜索、底部导航和页面留白规范
2. 修复历史乱码文案和不稳定的中文显示
3. 首页强化推荐商户、推荐商品和运营位
4. 完成发现页、详情页、购物车、订单、我的五个主页面的基础重构

## 第二阶段：交易链路

1. 增加购物车空状态、结算前置说明和金额汇总
2. 完善订单列表、状态分类和再次查看商品入口
3. 补齐商品详情中的加入购物车和立即购买动作反馈

## 第三阶段：账号和服务

1. 增加登录入口和游客态展示
2. 整理优惠券、积分、地址、客服等服务入口
3. 为后续登录注册和地址管理预留页面结构

## 第四阶段：生产化准备

1. 真机适配和移动端细节校验
2. 补齐加载态、空态、错误态
3. 配合宝塔部署进行构建、静态资源和 API 代理验证
